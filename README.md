# CLOMapping: ระบบการวัดและประเมินการบรรลุ CLOs และ PLOs

## 📌 สรุปภาพรวมโครงการ (Project Overview)
**CLOMapping** เป็นระบบที่ถูกออกแบบมาเพื่อเชื่อมโยงและประเมินผลการบรรลุผลลัพธ์การเรียนรู้ระดับรายวิชา (Course Learning Outcomes: CLOs) ไปสู่ผลลัพธ์การเรียนรู้ระดับหลักสูตร (Program Learning Outcomes: PLOs) 

โครงการนี้พัฒนาขึ้นโดยภาควิชาชีววิทยาป่าไม้ คณะวนศาสตร์ มหาวิทยาลัยเกษตรศาสตร์ เพื่อเชื่อมโยงสิ่งที่ไม่เคยเชื่อมต่อกันมาก่อน (Connecting the unconnected) จุดประสงค์หลักคือการประเมินผลโดยไม่เพิ่มภาระให้กับอาจารย์ประจำวิชา และอาจารย์ผู้รับผิดชอบหลักสูตรสามารถใช้งานได้โดยอาศัยเพียงข้อมูลเกรดของนิสิต ไม่จำเป็นต้องทราบคะแนนดิบ

**หมายเหตุ** สามารถใส่กิจกรรมอื่นที่ไม่ใช่รายวิชาลงไปได้ หากมีการประเมินผลเป็นระดับขั้น เช่น การตีพิมพ์ การนำเสนอผลงานทางวิชาการ ประสบการณ์การเป็นผู้ช่วยสอน เป็นต้น



## ✨ แนวคิดและกลไกการทำงาน (Key Concepts & Mechanics)
* **การประเมินตามระบบเกรด:** ระบบทำงานบนพื้นฐานของรายวิชา 15 สัปดาห์ และใช้ระบบเกรดปกติ (A, B, C) ในการวัดผลแบบอิงเกณฑ์
* **การแปลงผลการเรียนเป็นตัวคูณ (Grade Weighting):** ผลการเรียนของนิสิตจะถูกแปลงเป็นค่าน้ำหนักเพื่อนำไปคำนวณคะแนน PLO ดังนี้:
  * `A` = 1.00
  * `B+` = 0.75
  * `B` = 0.50
  * `C+` = 0.25
  * `<= C` = 0.00
* **การอ้างอิงนิสิตสมมุติ (Reference Student):** การวัดผลสัมฤทธิ์จะเปรียบเทียบกับ "นิสิตสมมุติ" (Ref) ที่ลงทะเบียนครบทุกวิชาในหลักสูตรและได้เกรด A ทั้งหมด ซึ่งจะทำให้ได้ตัวคูณ 1.0 ในทุกรายวิชา การประเมินผ่าน/ไม่ผ่าน จะใช้การหาจุดตัด (Cutoff) เช่น นิสิตต้องทำคะแนนให้ได้ร้อยละ 70 ของนิสิตสมมุติ
* **ข้อจำกัดเชิงวิเคราะห์:** นิสิตไม่จำเป็นต้องรับผลประเมินครบทุก PLO ระบบจะคำนวณเฉพาะผลลัพธ์ที่นิสิตทำได้จริง หากรายวิชารับความรับผิดชอบ PLO จำนวนมากเกินไป คะแนนประเมินจะถูกเจือจางลง ทำให้การมี PLO มากไม่ได้แปลว่าดีกว่าเสมอไป น้ำหนักของคะแนนในแต่ละวิชาจะเป็นไปตามจำนวนหน่วยการเรียนและสัดส่วนคะแนนการวัดผลของรายวิชานั้น

## 📂 โครงสร้างโฟลเดอร์และไฟล์ข้อมูลนำเข้า (Inputs & Directory Structure)
การประมวลผลของโปรแกรมจำเป็นต้องจัดเตรียมข้อมูลโดยแบ่งเป็นไฟล์และโฟลเดอร์หลักดังนี้:
* `credits/credits.csv`: ไฟล์เก็บข้อมูลจำนวนหน่วยกิตของแต่ละรายวิชา
* `@data/clomap/`: โฟลเดอร์สำหรับเก็บไฟล์ตารางเชื่อมโยง (Mapping) ระหว่างการวัดผลหรือกิจกรรมในห้องเรียน กับระดับ CLOs
* `@data/plomap/`: โฟลเดอร์สำหรับเก็บไฟล์ตารางเชื่อมโยงจากระดับ CLOs ไปยังระดับ PLOs
* `student_XX.csv`: ไฟล์ประวัติเกรดและผลการเรียนของนิสิตแต่ละคน

## 📊 ผลลัพธ์จากการประมวลผล (Outputs)
หลังจากประมวลผลคะแนนจากการเชื่อมโยงและคำนวณหน่วยกิตแล้ว ระบบจะส่งออกไฟล์ผลลัพธ์ในโฟลเดอร์ `@output/` ดังนี้:
* `plo_scores.csv`: ไฟล์ตารางแจกแจงคะแนน PLOs ของแต่ละรายวิชา (Course PLOs score distribution)
* `plo_contributions.csv`: ไฟล์ตารางแสดงสัดส่วนและน้ำหนักการมีส่วนร่วมในคะแนน PLOs ของแต่ละรายวิชา (Course PLOs score contributions)
* `score_student_XX.csv`: ไฟล์สรุปผลคะแนนการบรรลุ PLO ของนิสิตเป็นรายบุคคล เพื่อนำไปพิจารณาตามเกณฑ์ต่อไป
